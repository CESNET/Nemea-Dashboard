# QUERY API

All values are specified as URL parameters

## ?filter
This enables filtering and must be set if you want a filter query

## srcip <IP4>

Specify Source IP4 address to look for

## dstip <IP4>

Target IP4

## limit <int>

limit number of results

## dir <boolean>

direction of ordering of the results

* 1 - ascending
* -1 - descending

## orderby <string>

key to order by the results

## category <string>

fulltext search

## description <string>

fulltext search

## date <datetime>

javascript datetime

## from <hours:minutes>

* *Compulsory*
* Default value: 12:00

# to <hours:minutes>

specify window in which to search
